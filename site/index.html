As a little primer: you can make the "id" of an HTML element by declaring it's id inside of it's tag. This specifies a completely unique element.

These are edited in jQuery and CSS using the #idElement syntax.

you can make the "class" of an element, likewise, by declaring it's class, and edit it using the .class syntax.

Elements can have both id's and classes.
